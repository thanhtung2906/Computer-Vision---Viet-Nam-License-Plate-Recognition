# Computer-Vision---Viet-Nam-License-Plate-Recognition
This project features a high-performance Real-Time Traffic Monitoring pipeline designed for Automatic License Plate Recognition (ALPR).
The system integrates YOLOv8 for robust object detection, ByteTrack for persistent multi-object tracking, and PaddleOCR for precise character extraction from video streams.
To ensure high accuracy, the detection model was trained on a massive dataset of 23,000+ images, including 20,000 synthetic samples generated via Albumentations, achieving a solid 83% mAP@50. The entire pipeline is optimized for efficiency, maintaining a smooth inference speed of ~30 FPS, making it an ideal solution for real-world intelligent transportation systems.
